# react-native-windows-thermal-printer

Thermal Printer support for React Native Windows

## Installation

```sh
npm install react-native-windows-thermal-printer
```

## Usage

```js
import { multiply } from 'react-native-windows-thermal-printer';

// ...

const result = multiply(3, 7);
```
